# 📖 Overview

The **ESP Heart Rate Monitor** is an embedded IoT project designed to measure heart rate using a pulse sensor interfaced with an ESP8266 microcontroller.

The system continuously acquires pulse signals, processes the sensor data, and provides real-time heart rate monitoring for healthcare applications.

This project demonstrates biomedical sensor interfacing, embedded programming, and real-time signal acquisition.

# ⚙️ Working Principle

1. The pulse sensor detects changes in blood flow.
2. The ESP8266 reads the analog pulse signal.
3. The signal is processed to estimate heart rate.
4. The measured values are displayed or transmitted for monitoring.
5. The system continuously updates heart rate readings in real time.
