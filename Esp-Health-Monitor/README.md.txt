# 📖 Overview

The **ESP Health Monitor** is an embedded IoT project designed to monitor basic health parameters using a temperature sensor and an ESP8266 microcontroller.

The system continuously acquires sensor data, processes the readings, and provides real-time monitoring for healthcare applications.

This project demonstrates sensor interfacing, embedded programming, and IoT-based health monitoring.

# ⚙️ Working Principle

1. The DS18B20 sensor measures the temperature.
2. The ESP8266 reads the sensor data.
3. The measured value is processed by the microcontroller.
4. The system indicates the operating status using the onboard LED.
5. Temperature readings can be used for health monitoring and future IoT integration.
